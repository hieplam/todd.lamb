## Goal

Define and validate all content entry schemas for blog posts and static pages using Astro's Content Collections API with Zod, and expose typed loaders so the rest of the site can query entries with full TypeScript inference.

## Parent Fit

| Field | Value |
| --- | --- |
| Container | Hiep Lam Blog (c3-1) |
| Category | Foundation (01) |
| Owned files | src/content.config.ts, src/content/posts/, src/content/pages/ |
| Depended on by | pages-routing, layouts, ui-components |

## Purpose

Owns the Zod schema definitions for posts (fields: author, pubDatetime, modDatetime, title, featured, draft, tags, ogImage, description, canonicalURL, hideEditPost, timezone) and pages (title, description, ogImage, canonicalURL). Owns the raw markdown/MDX source files under src/content/. Does NOT own rendering, routing, or sorting logic.

## Foundational Flow

| Aspect | Detail | Reference |
| --- | --- | --- |
| Preconditions | Astro build syncs collections via astro sync before type-checking | c3-1 |
| Inputs | Raw .md/.mdx files in src/content/posts/ and src/content/pages/ matched by glob loader pattern [^_]*.{md,mdx} | c3-1 |
| State | Content collection registry in .astro/content.d.ts generated by astro sync at build time | c3-1 |
| Shared deps | src/config.ts for default author; astro/zod for schema validation; astro/loaders for glob loader | c3-1 |

## Business Flow

| Aspect | Detail | Reference |
| --- | --- | --- |
| Primary path | Build calls glob loader, entries parsed, Zod validates frontmatter, typed collection returned to callers | c3-1 |
| Drafts excluded | Posts with draft:true or future pubDatetime are excluded by postFilter utility before rendering | c3-1 |
| Underscore exclusion | Posts prefixed with underscore in filename are excluded by glob pattern [^_]* to allow private drafts | c3-1 |
| Failure | Zod validation error at build time stops the build with a schema mismatch message surfaced by astro check | c3-1 |

## Change Safety

| Risk | Trigger | Detection | Required Verification |
| --- | --- | --- | --- |
| Schema field rename breaks existing posts | Adding or removing required frontmatter field from Zod schema | astro check reports TypeScript type errors at build | src/content.config.ts change + astro check output |
| Glob pattern excludes valid posts | Changing the [^_]* file exclusion pattern in the glob loader | Missing posts in built dist/ output | src/content.config.ts change + verify post count in dist/ |
